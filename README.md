# Open-Source Redirector

Program to redirect your from proprietary web-services to their open-source alternative.

Redirect List:

-   YouTube ➝ Invidious
-   YouTube Music ➝ Hyperpipe
-   Google ➝ SearX
-   Google Translate ➝ Lingva Translate
-   Yahoo ➝ SearX
-   Bing ➝ SearX
-   Reddit ➝ Libreddit
-   Twitter ➝ Nitter
-   Wikipedia ➝ Wikiless
-   Medium ➝ Scribe
-   TikTok ➝ ProxiTok
